# Dice Roller

Dice Roller App implemented on Electron.

## Instructions

To start this app:

- Installing Dependencies:

```
npm ci
```

- Compiles and hot-reloads for development:

```
npm run start
```

- Compiles and minifies for production:

```
npm run package
```

- Run unit tests:

```
npm run test:unit
```

- Run lint:

```
npm run lint
```

- Check format files:

```
npm run format
```
