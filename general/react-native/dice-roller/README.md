# Dice Roller

Dice Roller App implemented on React Native.

## Instructions

To start this app:

- Installing Dependencies:

```
npm ci
```

- Start Metro dev server:

```
npm run start
```

- Build and Run app:

With Metro running, open a new terminal and execute:

```
npm run android
```

- Run unit tests:

```
npm run test:unit
```

- Run lint:

```
npm run lint
```

- Check format files:

```
npm run format
```
